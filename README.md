# Minecraft Server Downloader
## Install Required Python Modules
### 

### Windows/macOS/Linux
  ```sh
    pip3 install -r requirements.txt
  ```
or 
  ```sh
    pip install -r requirements.txt
  ```
## Run MC Server Downloader  
CLI:      
```python CLI.py```  

# Dependencies for Linux  
## Arch  
```sudo pacman -S tk```  
## Debian  
```sudo apt-get install python-tk```
